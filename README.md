# Análisis de Ventas de una Pequeña Empresa

## 🧭 Visión general
Este proyecto forma parte del trabajo práctico sobre gestión colaborativa, desarrollado por un equipo que utiliza herramientas como Jira, Git, GitHub y Google Colab para aplicar buenas prácticas de trazabilidad y control de versiones.

El objetivo principal es analizar datos simulados de ventas comerciales para generar indicadores básicos que permitan interpretar el desempeño de una pequeña empresa.

## 🎯 Objetivo del análisis
Analizar la información de ventas y obtener métricas clave que reflejen el rendimiento del negocio.

**Indicadores esperados:**
- Ventas totales
- Producto más vendido
- Evolución de ventas por mes
